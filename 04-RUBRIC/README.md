# 04 — Rubric

The 22-principle PaaP rubric: definitions, archetype-aware scoring, empirical grounding, and reflexive validation.

## What's in this folder

| File | Purpose |
|---|---|
| [`principles.md`](./principles.md) | The 22 principles in full — name, definition, archetype applicability, scoring guidance |
| [`scoring-template.md`](./scoring-template.md) | Blank rubric template for scoring a skill end-to-end |
| [`empirical-validation.md`](./empirical-validation.md) | N=50 community-skill survey — rubric calibration against real-world skills |
| [`corpus-50-skills.md`](./corpus-50-skills.md) | Corpus index: 50 community skills surveyed in v0.1 |
| [`corpus-extension-30-skills.md`](./corpus-extension-30-skills.md) | v0.2 extension: 30 additional skills (total N=80, 36 distinct authors) |
| [`reflexive-self-audit.md`](./reflexive-self-audit.md) | The repo scored against its own rubric — eating our own dogfood |

## How to use

- **Authoring a skill?** Start with [`principles.md`](./principles.md), score with [`scoring-template.md`](./scoring-template.md).
- **Evaluating a skill?** Use [`/paap-eval`](../06-META-PAAP/paap-eval/) for auto-scoring, or apply [`scoring-template.md`](./scoring-template.md) manually.
- **Validating the rubric?** Read [`empirical-validation.md`](./empirical-validation.md) and the corpus extension.

See also: [03-ANATOMY/](../03-ANATOMY/) for the structural skeleton each principle attaches to.
